# Boutique React

Projet universitaire de boutique en ligne développé avec React dans le cadre de ma formation en informatique.

L'application permet d'afficher un catalogue de produits, de rechercher un article, d'ajouter des produits dans un panier, de modifier les quantités, de supprimer des articles et de calculer automatiquement le total de la commande.

## Démo en ligne

https://christianmahi.github.io/Boutique-React/

## Fonctionnalités

* Affichage d'un catalogue de produits
* Recherche de produits par nom
* Ajout d'un produit au panier
* Suppression d'un produit du panier
* Modification des quantités dans le panier
* Mise à jour du stock disponible
* Calcul du poids total
* Calcul du prix total de la commande
* Affichage dynamique avec React

## Technologies utilisées

* React
* JavaScript
* HTML
* CSS
* Webpack
* GitHub Pages

## Objectif du projet

Ce projet m'a permis de travailler les bases de React à travers une application concrète.

J'ai notamment manipulé les composants, les props, l'état, les événements utilisateur et la mise à jour dynamique de l'interface en fonction des actions réalisées dans le panier.

## Lancer le projet en local

Cloner le projet :

```bash
git clone https://github.com/christianmahi/Boutique-React.git
cd Boutique-React
```

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev-server
```

Créer la version de production :

```bash
npm run build
```
